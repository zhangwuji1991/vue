### 实现的功能
1. 首页轮播图
2. 新闻实时请求、点击加载更多
3. 请求新闻时加载动画
4. 菜单栏的显示和隐藏
5. 搜索界面
6. vuex管理数据

### 未实现的功能
* 搜索

## Build Setup

``` bash
# 下载或克隆下来，然后安装依赖
npm install

# 开发预览
npm run dev

# 打包发布，生成的文件在dist文件夹中
npm run build
```

## 关键目录介绍
```
src： 项目代码
   assets: 项目中静态文件，如图片
   components: vue组件
   router: 路由文件
   store： 项目数据
   main.js： 入口文件
```
